# p5-test

A testing project exploring the capabilities of p5.js and ml5.js for image classification and object detection.

## Overview

This project implements various machine learning features using ml5.js, including:
- Image Classification
- Object Detection
- Detection Zones with Statistics

## Features

- Real-time video classification using MobileNet
- Object detection using COCO-SSD
- Custom detection zones with statistics
- Data export functionality
- Configurable confidence thresholds
- FPS counter and performance monitoring

## Dependencies

- p5.js (v1.4.0)
- ml5.js (v0.12.2)

## Setup

1. Clone the repository
2. Open any of the HTML files in a modern web browser
3. Grant camera permissions when prompted
